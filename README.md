# What Is My IP address

Prints your IP address, using the [httpbin](https://httpbin.org/) service.

## Usage

    wimip
    60.242.200.42

## Install

    go get -u github.com/scottjbarr/wimip

## References

- [httpbin(1): HTTP Request & Response Service](https://httpbin.org/)
